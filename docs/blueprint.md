# **App Name**: Itaaf

## Core Features:

- Product Catalog: Display products with images, descriptions, and prices.
- Shopping Cart: Allow users to add, remove, and update items in their cart.
- Checkout Process: Guide users through secure checkout with payment gateway integration.
- Order Management: Allow users to view their order history and order details.
- AI Product Recommendations: Suggest related products based on viewing history to increase sales; the LLM will act as a tool for incorporating customer interest, popularity, inventory status and seasonal sales data to personalize results.
- User Authentication: Secure user registration and login.

## Style Guidelines:

- Primary color: Calm blue (#79A8DC), suggestive of peace and trust.
- Background color: Very light blue (#E8F0FE) for a clean, airy feel.
- Accent color: Muted violet (#B18CD9) for a subtle contrast that maintains a sense of calm.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text
- Use soft, rounded icons to enhance the friendly feel.
- Subtle transitions and animations for a smooth user experience.
- Clean, spacious layout with ample whitespace to create a relaxing browsing experience.
